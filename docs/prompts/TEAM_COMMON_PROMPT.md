```text
이 저장소는 4명이 Codex로 병렬 구현하는 Vanilla JavaScript 프로젝트다.

반드시 먼저 아래 문서를 읽어라.

공통 규칙:
- 최우선 목표는 기능이 실제로 동작하는 결과를 만드는 것이다.
- 나는 네가 허용된 파일만 수정하길 원한다.
- 공통 고정 파일은 절대 수정하지 마라.
- 새로운 파일을 만들지 마라.
- 폴더 구조를 바꾸지 마라.
- 다른 역할의 파일은 읽을 수만 있고 수정하면 안 된다.
- 계약이 안 맞거나 충돌 위험이 보이면 임의로 우회 구현하지 말고 멈춘 뒤 보고해라.
- 파일 경계를 지키면서도, 네가 맡은 범위 안에서는 기능 완성을 끝까지 책임져라.
- 서버 코드, 배포 코드, 라이브러리 추가는 금지다.
- 이 프로젝트는 로컬 브라우저에서 동작하는 클라이언트 사이드 구현만 하면 된다.
- 구현은 JavaScript 중심으로 진행하고, HTML/CSS는 정해진 파일 범위에서만 다뤄라.

작업 방식:
- 먼저 네가 수정 가능한 파일 목록을 다시 적어라.
- 그 다음 해당 파일만 읽고 수정해라.
- 작업 중 다른 파일 수정이 필요해 보이더라도 손대지 마라.
- 완료 후에는 "수정한 파일 목록"과 "건드리지 않은 파일 경계"를 함께 보고해라.
```
